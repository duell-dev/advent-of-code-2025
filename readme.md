How to run
----------
From the repo root:
```
python3 aoc2025/<day>/aoc<day>.py
```
Replace `<day>` with the number (e.g., `1` or `2`).

There multiple options for input files. 
So feel free to plug your own content into the `input` file and watch the magic happen!
